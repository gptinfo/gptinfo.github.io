2. 개인정보의 이용 목적

  • 위치 기반 맛집/음식점 검색 서비스 제공
  • 음성 인식을 통한 검색 기능 제공
  • 텍스트 인식(OCR) 기능 제공
  • 맞춤형 광고 제공 (Google AdMob, AppLovin)
  • 푸시 알림 발송 (Firebase Cloud Messaging)
  • 서비스 개선 및 이용 통계 분석

4. 보관 기간 및 파기

  • 로컬 데이터 (AsyncStorage, SQLite): 앱 삭제 시 자동 파기
  • 서버 데이터: 삭제 요청 시 파기
  • 광고 관련 데이터: 각 SDK 제공사의 보관 정책에 따름

5. 이용자의 권리

  • 위치정보 수집 동의/철회 (기기 설정)
  • 카메라/마이크 권한 관리 (기기 설정)
  • 푸시 알림 수신 동의/철회
  • 광고 추적 거부 (기기 광고 설정)
  • 데이터 삭제 요청: gptinfo@gmail.com

6. 아동의 개인정보 보호

앱에 유아 콘텐츠가 포함되어 있으나, 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 보호자의 동의 없이 아동의 개인정보가 수집된 것을 인지한 경우, 즉시 해당 정보를 삭제합니다.

7. 개인정보처리방침 변경

본 방침이 변경되는 경우, 앱 내 공지를 통해 안내합니다.
